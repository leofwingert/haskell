# Jogo Pedra-Papel-Tesoura-Fogo-Água

Este é um jogo simples implementado em Haskell.

## Como rodar

Para compilar e rodar o jogo, siga os passos abaixo:

1. Instale a dependência necessária:
   ```bash
   cabal install --lib random
   ```

2. Compile o código:
   ```bash
   ghc jogo.hs -o jogo
   ```

3. Execute o jogo:

   - **Linux/macOS:**
     ```bash
     ./jogo
     ```

   - **Windows:**
     ```powershell
     .\jogo.exe
     ```
